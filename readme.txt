== Description ==

This plugin helps you to set up a shortcut key for the comments submiting. Such as 'Ctrl + Enter'.

**Features:**

* Support for WordPress 2.5
* Shortcut key for comments
* Custom shortcut key
* Support various broswers

**Supported Languages:**

* English (default, 1.0 or higher)

**Demo:**

<a href="http://www.gekimoe.org/message-board#comment">http://www.gekimoe.org/message-board#comment</a>

**Notice**

If you installed an older version before, please deactive it first and reactive the newer version.
This may help you to clear up the depreacted options.

== Installation ==

1. Unzip archive to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make some changes in the 'setting' => 'WP Comments Shortcut' to meet your preferences
